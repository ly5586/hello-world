# hello-world

A friendly CLI tool that greets you by name in 12 different languages.

## Features

- **12 Languages**: Chinese, English, Japanese, Korean, Spanish, French, German, Italian, Portuguese, Russian, Arabic, Hindi
- **Colorful Output**: Each language rendered in its own terminal color
- **Zero Config**: Just install and run
- **Beginner Friendly**: Well-commented source code, perfect for learning CLI tool development

## Quick Start

```bash
npm install -g hello-world
hello-world --name "小明"
```

## Usage

```
hello-world [options]

Options:
  -n, --name <name>    Your name (default: "World")
  -l, --lang <lang>    Single language output (e.g. "ja", "ko")
  --list               List all supported languages
  -v, --version        Show version
```
